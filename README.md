# 💳 Credit Card Fraud Detection System

This project is a Machine Learning-based system that detects fraudulent credit card transactions.

---

## 🚀 Features

- 🔍 Detects fraud transactions using ML model
- 🎲 Random transaction testing
- ⚠️ Force fraud testing
- 💻 Interactive dashboard using Streamlit
- 📊 Displays prediction with confidence score

---

## 🧠 Machine Learning

- Model: Random Forest Classifier
- Dataset: Highly imbalanced fraud dataset
- Techniques used:
  - Data balancing (resampling)
  - Train-test split
  - Model evaluation (Precision, Recall, F1-score)

---

## 📊 Dataset

The dataset used is from Kaggle:

👉 https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

⚠️ Note: Dataset is not included in this repository due to size limitations.
